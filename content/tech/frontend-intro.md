# 前端开发入门

前端开发是当前最热门的技术方向之一，本文将为你规划一条清晰的学习路线。

## 学习路线

### 第一阶段：基础三件套

#### HTML

HTML 是网页的骨架，掌握常用的标签和语义化：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>我的第一个网页</title>
</head>
<body>
  <h1>Hello HDU!</h1>
</body>
</html>
```

#### CSS

CSS 负责网页的样式和布局：

- 盒模型
- Flexbox 布局
- Grid 布局
- 响应式设计（媒体查询）
- CSS 动画

#### JavaScript

JavaScript 是前端的核心编程语言：

- 变量、数据类型、运算符
- 函数、作用域、闭包
- DOM 操作
- 事件处理
- ES6+ 新特性

### 第二阶段：框架学习

推荐学习 React 或 Vue：

- **React**：生态丰富，就业面广
- **Vue**：上手简单，国内使用广泛

### 第三阶段：工程化

- Git 版本管理
- npm/pnpm 包管理
- Webpack/Vite 构建工具
- TypeScript 类型系统
