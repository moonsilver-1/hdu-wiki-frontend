# 竞赛入门指南

HDU 在编程竞赛方面有着悠久的传统和优秀的成绩，本文将帮助你快速入门。

## 竞赛类型

### ACM-ICPC

ACM 国际大学生程序设计竞赛是最具影响力的编程竞赛。

- **参赛形式**：3 人组队，5 小时，共用 1 台电脑
- **语言**：C/C++、Java、Python
- **重点**：算法、数据结构、数学、图论

### 蓝桥杯

国内影响力最大的编程竞赛之一。

- **参赛形式**：个人赛
- **组别**：C/C++、Java、Python 等
- **重点**：基础算法、搜索、动态规划

### 其他竞赛

- 中国大学生程序设计竞赛（CCPC）
- 百度之星
- LeetCode 周赛/双周赛

## 如何入门

### 第一步：打好基础

```
学习顺序：
C语言 → 数据结构 → 基础算法 → 进阶算法
```

### 第二步：开始刷题

1. 注册 [HDU OJ](https://acm.hdu.edu.cn/)
2. 从简单的题目开始（HDOJ 1000-1099）
3. 每天坚持 1-2 题
4. 学会看题解但不要直接抄

### 第三步：系统学习

推荐学习路线：

1. **基础算法**：排序、二分、贪心
2. **搜索**：BFS、DFS、回溯
3. **动态规划**：背包、LIS、LCS
4. **图论**：最短路、最小生成树
5. **数学**：数论、组合数学
