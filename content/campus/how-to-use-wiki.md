## 欢迎

你现在已经打开了 [hdu-wiki.cn](https://hdu-wiki.cn)，恭喜你，你已经完成了最困难的一步——找到这个网站。

接下来我会教你怎么用这个东西。别担心，它比你想象的简单。

## 普通模式（推荐正常人使用）

打开网站首页，你会看到几个分类卡片：课程与学术、校园生活、技术与项目、社团与活动。点进去，选文章，读文章。

就这么简单。不需要教程对吧？

但既然你都点进来了，我还是多说两句：

- **搜索**：按 `Ctrl+K`，输入关键词，回车。就这么用。
- **目录**：打开文章后右侧有目录导航，点击可以快速跳转到对应章节。
- **标签**：文章底部有标签，点击可以...好吧目前点击标签还不能筛选，但视觉上很好看。

## Vim 模式（给那些想在浏览器里装酷的人）

你可能在导航栏看到了一个叫 **Vim** 的链接。点进去，你的整个浏览器会变成一个看起来像 Neovim 的界面。

这不是 Bug，这是 Feature。

### 为什么要搞一个 Vim 模式？

好问题。理由有三：

1. **沉浸式阅读**：没有花里胡哨的导航栏和侧边栏，你的注意力只会在文字上。就像在终端里看文档一样纯粹。
2. **键盘流**：不用摸鼠标，双手不离键盘，文章从头刷到尾。效率选手狂喜。
3. **帅**：当你室友走过看到你在浏览器里用 Vim 看文章时，他们会觉得你是个黑客。虽然你只是在看「如何安装软件」这种入门文章。

### 基本操作

打开 Vim 模式后，左侧是文件树（NeoTree），右侧是文章内容（Buffer）。

**导航：**

| 按键 | 作用 |
|------|------|
| `j` / `k` | 上下移动 |
| `l` | 在文件树中打开目录或文章 |
| `h` | 在文件树中折叠目录 |
| `gg` | 跳到文章开头 |
| `G` | 跳到文章结尾 |
| `Ctrl+d` / `Ctrl+u` | 向下/向上翻半页 |

**切换面板：**

| 按键 | 作用 |
|------|------|
| `Space w` 或 `Space e` | 在文件树和文章之间切换焦点 |
| `Shift+L` | 切换到下一个已打开的文章 |
| `Shift+H` | 切换到上一个已打开的文章 |

**搜索：**

| 按键 | 作用 |
|------|------|
| `/关键词` | 向下搜索 |
| `?关键词` | 向上搜索 |
| `n` / `N` | 跳到下一个/上一个匹配 |

**命令模式：**

按 `:` 进入命令模式，支持的命令：

| 命令 | 作用 |
|------|------|
| `:q` | 退出 Vim 模式，回到普通网站 |
| `:e tech/how-to-use-llm` | 直接打开指定文章 |
| `:help` | 显示帮助（和这篇差不多但没这么有趣） |
| `:ls` | 查看已打开的文章列表 |
| `:bn` / `:bp` | 下一个/上一个文章 |

按 `F1` 也可以随时查看快捷键参考。

### 一个典型的操作流程

1. 按 `Space w` 跳到文件树
2. 用 `j` `k` 找到想看的分类，按 `l` 展开
3. 继续用 `j` `k` 选文章，按 `l` 打开
4. 自动跳转到文章内容，用 `j` `k` 阅读，`Ctrl+d` `Ctrl+u` 翻页
5. 按 `Space w` 回到文件树选下一篇
6. 不想看了，按 `:q` 回到正常世界

## 最后

不管你用普通模式还是 Vim 模式，内容都是一样的。选你用着舒服的方式就好。

如果你用了 Vim 模式并且觉得它很酷，欢迎在同学面前炫耀。如果你用了之后一脸懵逼，那也完全没问题——普通模式就在那里，它不会嫉妒的。
